The problem

Job seekers often don't know how their resume reads to an Applicant Tracking System, or how they'd hold up in a real interview for a specific role. This agent gives feedback on both in one place.

What it does

  • Analyzes an uploaded resume the way an ATS would score it
  • Runs role-based mock interviews tailored to a target job
  • Generates personalized career recommendations from the analysis
  • Visualizes results and progress with interactive charts

How it's built

A Flask (Python) backend handles resume parsing with NLTK and stores data in SQLite through SQLAlchemy. The front-end is built with HTML, CSS, and JavaScript, with Chart.js rendering the analysis results, all connected through REST APIs.
