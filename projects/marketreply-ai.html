The problem

Sellers on online marketplaces get flooded with buyer negotiation messages, and manually checking each one against pricing rules is slow and inconsistent. MarketReply AI automates that first pass.

What it does

  • Reads incoming buyer messages and checks them against seller-defined pricing and negotiation rules
  • Uses the Google Gemini API to auto-draft a context-aware reply
  • Supports multiple sellers with JWT-secured, isolated accounts
  • Maintains a live, browsable seller directory

How it's built

The front-end is a React app built with Vite and styled with Tailwind CSS. The backend is a Spring Boot (Java) service backed by MongoDB Atlas, with JWT authentication securing each seller's account and REST APIs connecting the two layers.
