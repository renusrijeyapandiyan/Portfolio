The problem

Training and testing a natural-language-understanding model usually means juggling notebooks, scripts, and separate dashboards. BotTrainer pulls that workflow into a single interactive web app, so queries can be entered, labelled, and evaluated in one place.

What it does

  • Organizes projects into workspaces (e.g. "student bot", "healthcarebot")
  • Accepts CSV, JSON, or YML datasets, up to 100MB, and summarizes rows/columns automatically
  • Trains multiple supervised algorithms in one run — Random Forest, XGBoost, SVM, Logistic Regression, Decision Tree, KNN, Naive Bayes
  • Serves a RASA-powered chat interface that shows detected intent and confidence score per message

How it's built

The front-end is a Next.js and React application written in TypeScript. Data is modelled with Drizzle ORM on top of Supabase, with a Node.js layer handling REST endpoints, and a Python service handling the NLU processing itself.
